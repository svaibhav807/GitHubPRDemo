# GitHubPRDemo

GitHub mobile application clone built in Swift using MVVM. 



To get it up and running - 
- clone
- pod install
- open MyGitHub.xcworkspace

Time Taken - 17 Hours. 

What is done:
- Pagination
- Title, Created date, Closed date, userName, User Image
- Error Handling
- Pull to refresh
